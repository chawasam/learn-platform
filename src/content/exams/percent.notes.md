## Archetype ที่ออกจริง (สอบเข้า ม.1 / O-NET ป.6)
ร้อยละออกข้อสอบเข้า ม.1 แทบทุกปี แนวหลัก:
1. หาค่าร้อยละของจำนวน (p% ของ N)
2. หาจำนวนเมื่อรู้ร้อยละ (รู้ p% = v → หา N)
3. เป็นร้อยละเท่าไรของ (A เป็นกี่ % ของ B)
4. ส่วนลด — ลดกี่บาท / จ่ายจริง / **ลดซ้อนสองชั้น**
5. กำไร / ขาดทุน — หาราคาขาย / หาต้นทุน / กำไรร้อยละ
6. ดอกเบี้ย (ต่อปี, หลายปี)
7. ภาษีมูลค่าเพิ่ม (VAT 7%)
8. เพิ่มขึ้น/ลดลงเป็นร้อยละ
9. โจทย์ปัญหาประยุกต์ (เทียบบัญญัติไตรยางศ์)

## Distractor pattern (เด็กพลาดบ่อย → ใช้เป็นตัวเลือกลวง)
- ตอบ "ส่วนลด" แทน "ราคาจ่ายจริง" (และกลับกัน)
- ลืมหาร 100 (ตอบเลข % ดิบ)
- หาต้นทุนด้วยการลบ % จากราคาขาย (แทนการหาร 1+p%)
- ลดซ้อน: เอา % มารวมกัน (25%+10%=35%) แทนคิดทีละชั้น
